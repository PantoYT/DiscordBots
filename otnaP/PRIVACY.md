# Privacy Policy — otnaP

Last updated: 2026-06-02

## What we collect
- otnaP collects **no persistent user data**
- Discord user IDs and guild IDs are used only in-memory during runtime and are never written to disk

## /ask command
When you use `/ask`, your query text is sent to the OpenRouter API to generate a routing suggestion. OpenRouter may log requests according to their own privacy policy. otnaP does not store the query or the response.

## /status command
Bot presence data is read from Discord's gateway in real time. Nothing is stored.

## What we do NOT collect
- Usernames, avatars, email addresses
- Message content outside of slash command interactions
- Any data that persists between bot restarts

## Data deletion
There is no stored data to delete. If you have concerns about OpenRouter data, refer to [openrouter.ai/privacy](https://openrouter.ai/privacy).
