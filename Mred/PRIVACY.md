# Privacy Policy — Mred

Last updated: 2026-06-02

## What we collect
Mred collects **no personal data**. No user information is stored.

## Third-party API
Mred communicates with the Spotify Web API using a server-side Client Credentials token. Queries contain only search terms — no user identity is sent to Spotify.

## Data storage
Mred has no persistent storage. No files are written to disk during operation.

## Data deletion
There is no stored data to delete.
